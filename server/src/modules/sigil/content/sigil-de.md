# 1. Bundeslandshintergrund

Der Bundeslandshintergrund zeigt, **in welchem Bundesland die Stadt liegt**.

## Vorgehen

1. Ausgangspunkt ist die **Flagge des Bundeslandes**.
2. Die Flagge wird **vereinfacht**:
   - Wappen werden entfernt.
   - komplizierte Muster werden reduziert.
3. Die Farben der Flagge werden, außer bei Hamburg, als **horizontale Streifen** in einen Kreis übertragen.

## Beispiel

Baden-Württemberg → oben schwarz, unten gelb
Niedersachsen → schwarz, rot, gelb
Schleswig-Holstein → blau, weiß, rot

<!-- SECTION 2 -->
# 2. Bevölkerungsrahmen

Der Bevölkerungsrahmen zeigt die **Größe der Stadt** anhand der Einwohnerzahl.

| Einwohnerzahl | Klasse | Rahmenart |
|-----|-----|-----|
| bis 500.000 | Mittelstadt | gleichmäßiger grüner Rand |
| 500.001 - 1.000.000 | Großstadt | vier Markierungen am Rand (oben, unten, links, rechts) |
| über 1.000.000 | Megastadt | vier Markierungen + deutlich breiterer grüner Rand

## Beispiel

Münster (316.403 Einwohner) → Mittelstadt
Dortmund (587.696 Einwohner) → Großstadt
Berlin (3.664.088 Einwohner) → Megastadt

<!-- SECTION 3 -->
# 3. Hauptstadtkrone

Manche Städte erhalten eine **Hauptstadtkrone oben innerhalb des Siegels**.

Die Hauptstadtkrone zeigt den **politischen Status der Stadt**.

| Status | Darstellung |
|-----|-----|
| Bundeshauptstadt | goldene Linienkrone, darunter ein roter und ein goldener Balken |
| Landeshauptstadt | goldene Linienkrone, darunter ein roter Balken |
| ehemalige Bundeshauptstadt Bonn | goldene Linienkrone |
| kein Hauptstadtstatus | keine Hauptstadtkrone |

## Beispiel

Berlin → Bundeshauptstadt-Krone
München → Landeshauptstadt-Krone
Bonn → Sonderfall-Krone
Köln → keine Krone

# 4. Orientierungskreis (Lage im Bundesland)

Im unteren Bereich des Siegels befindet sich der **Orientierungskreis**.

Er enthält einen **Lagekreis** — einen goldenen Punkt, der zeigt, **wo die Stadt innerhalb ihres Bundeslandes liegt**.

## Himmelsrichtungen

| Lage | Position des Lagekreises |
|-----|-----|
| Norden | oben |
| Süden | unten |
| Osten | rechts |
| Westen | links |
| Nordost | oben rechts |
| Nordwest | oben links |
| Südost | unten rechts |
| Südwest | unten links |
| Mitte | mittig |

## Stadtstaaten

Für Stadtstaaten (Berlin, Hamburg, Bremen) gilt eine Sonderregel:

- der Lagekreis ist **größer**
- er liegt **zentral**

Dies zeigt, dass Stadt und Bundesland **identisch sind**.

## Beispiel

München (Lage: Süd) → Lagekreis unten
Dresden (Lage: Ost) → Lagekreis rechts
Hamburg (Stadtstaat) → großer Lagekreis zentral

<!-- SECTION 5 -->
# 5. Gründungsmittelpunkt

In der Mitte des Siegels befindet sich der **Gründungsmittelpunkt** — ein **schwarzer Kreis mit einer Zahl**.

Diese Zahl zeigt das **Jahrhundert der Stadtgründung**.

Das Gründungsjahr wird auf das **nächste Jahrhundert aufgerundet**.

## Beispiel

| Gründungsjahr | Anzeige |
|-----|-----|
| 1158 | 12 |
| 1299 | 13 |
| 700 | 7 |

Die Zahl steht **in goldener Schrift im Gründungsmittelpunkt**.

<!-- SECTION 6 -->
# 6. Kurzkennzeichen

Links im Siegel wird das **Kurzkennzeichen** (Kfz-Kennzeichen der Stadt) angezeigt.

## Darstellung

- rechteckiges Feld
- links ein **blauer Streifen**
- rechts ein **weißes Feld**
- darin das Kurzkennzeichen in schwarzer Schrift

## Beispiel

München → **M**

<!-- SECTION 7 -->
# 7. Koordinatenrechteck

Rechts im Siegel steht das **Koordinatenrechteck** mit geografischen Koordinaten.

## Darstellung

- weißes Rechteck
- hellblauer Rahmen
- Format: **Breitengrad|Längengrad** (ganzzahlig gekürzt, getrennt durch einen senkrechten Strich)

## Beispiel

München (48.1371° N, 11.5755° E) → **48|11**
